# Setup

- [NodeJS](https://nodejs.org/en/download/)
- [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

Execute in terminal:
> node -v

> npm -v

> npm init -y

> npm install --save-dev jest

# Run tests written in page-object-model
> npm run test
